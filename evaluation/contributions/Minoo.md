# Minoo – Contributions

## Features Implemented
I worked on the **Frontend V1** of the vehicle rental application.

### Frontend Implementation
I implemented the main frontend features using **React, useState, fetch, and React Router**.

The user can:
- view all vehicle rentals
- view details of a specific rental
- create a new vehicle rental
- update an existing vehicle rental
- delete a vehicle rental

### Pages and Components
I implemented the following pages and components:

- Navbar navigation
- Vehicle rental listings
- Vehicle rental details page
- Add vehicle rental page
- Edit vehicle rental page
- Not found page

### API Integration
I connected the frontend to the backend API.

- Used **fetch** to communicate with backend endpoints
- Implemented **GET, POST, PUT, and DELETE requests**
- Configured **Vite proxy** to connect the frontend with the deployed backend on Render
